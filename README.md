## 📝 README.md

```markdown
# NovaSphere

## 🌐 Project Overview
**NovaSphere** is a multipage portfolio and creative showcase website designed to highlight interactive design, animations, and responsive layouts. This project demonstrates proficiency in **HTML5, CSS3, and JavaScript**, combining visual appeal with functional interactivity.

---

## 📄 Pages
1. **Home**: Hero section with introduction and CTA.  
2. **About**: Overview of NovaSphere and mission.  
3. **Services**: Description of offered services with cards.  
4. **Gallery**: Showcase of projects using high-quality images.  
5. **Contact**: Contact form with validation and social links.

---

## 🎨 Features
- **Responsive Design**: Works on mobile, tablet, and desktop.  
- **CSS Animations**: Hero fade-in, card hover, gallery zoom.  
- **JavaScript Interactivity**:
  - Mobile navigation toggle  
  - Dark mode toggle (🌙 / ☀️) with persistence  
  - Contact form validation with alerts  
- **Gallery**: High-quality Unsplash images integrated.  
- **Clean Code**: Organized folders (`/css`, `/js`) and meaningful file names.  

---

## 🛠 Technologies Used
- **HTML5**: Semantic structure and accessibility  
- **CSS3**: Flexbox, Grid, transitions, animations, dark mode  
- **JavaScript**: DOM manipulation, functions, event listeners, localStorage  
- **Images**: Unsplash (free, high-resolution)  

---
```
